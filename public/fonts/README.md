# Reading fonts (self-hosted)

Fetched by `webgen/fetch_fonts.py` from Google Fonts, all under the SIL Open Font
License (OFL) — free to redistribute, so committed here for CSP-safe self-hosting
(no CDN at build or runtime). @font-face rules: `src/styles/reading-fonts.css`.

- **Sorts Mill Goudy** — body/running text (Goudy Old Style revival; the WoD20 corpus body face)
- **Cinzel** — Mago display/titles (echoes Abbess)
- **Marcellus** — Vampiro & Cazador display (echoes Friz Quadrata)
- **Metamorphous** — Hombre Lobo display (echoes Garou)
- **IM Fell English** — Wraith display (echoes Matrix Tall)
- **Uncial Antiqua** — Changeling display (echoes Kells)
